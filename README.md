# NLSE Primer

**Language / 语言**: [English](#english) | [中文](#中文)

## English

This repository contains LaTeX source files for a primer on the Nonlinear Schrödinger Equation (NLSE), which is essential for modeling nonlinear optical phenomena. It aims to provide intuitive explanations of various concepts and phenomena for graduate students or advanced undergraduates. It also links to various useful resources.

### LaTeX File Structure

- **Raw_TeX_English/**: English version LaTeX files
  - `main.tex`: Main document file
  - `ch-*.tex`: Individual chapter files covering topics like dispersion, solitons, supercontinuum, etc.
  - `AcronymList.tex` & `Acronyms.tex`: Acronym definitions

- **Raw_TeX_Chinese/**: Chinese version LaTeX files (same structure as English version)
  - `main.tex`: 主文档文件
  - `ch-*.tex`: 各章节文件，涵盖色散、孤子、超连续谱等主题
  - `AcronymList.tex` & `Acronyms.tex`: 缩写词定义



## 中文

此资源库包含汉化的非线性薛定谔方程入门教程的 LaTeX 源文件，该入门教程对非线性光学现象建模至关重要。它旨在为研究生或高年级本科生提供不同概念和现象的直观解释。它还链接了各种有用的资源。

- **Raw_TeX_English/**: 英文版 LaTeX 文件

  - `AcronymList.tex` 与 `Acronyms.tex`: 缩写词定义

- **Raw_TeX_Chinese/**: 中文版 LaTeX 文件（结构与英文版相同）
  - `main.tex`: 主文档文件
  - `ch-*.tex`: 各章节文件，涵盖色散、孤子、超连续谱等主题
