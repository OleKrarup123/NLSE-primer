\chapter{超连续谱产生}
\label{ch:supercontinuum}

前几章分别介绍了单独的线性与非线性效应，以及少数几种效应之间的相互作用。表~\ref{tab:NL_summary} 对它们的影响做了一个总结。对于某一介质，如果高功率脉冲满足载波频率处 $\betag_2\lesssim 0$、$\gamma>0$，且脉冲持续时间短于约 100~fs，那么表中列出的所有效应都可能同时存在。在这种情况下，脉冲演化往往会极其复杂，并可能使其频谱拓宽到初始带宽的十倍甚至二十倍~\cite{supercontinuum_original_paper}。本章给出一个超连续谱产生的例子，并说明这一过程如何由前文介绍的各类效应共同解释。

\section{案例分析}
为了模拟超连续谱的产生，采用了表~\ref{tab:SC_params} 中的参数。得到的超连续谱如图~\ref{fig:SC_combined} 所示。图~\ref{fig:SC_combined}c) 中的频谱表明：在传播前 1~m 内，脉冲主要经历 SPM；而图~\ref{fig:SC_combined}b) 表明紧接着发生了孤子裂变。那个功率较高、持续时间更短、并沿抛物线轨迹向后时刻“走离”的脉冲，是一个不断自发红移的拉曼孤子；而那个功率较低、沿近似线性轨迹向后走离的光，则是一支频率约为 320~THz 的色散波。关于这一超连续谱特性的详细分析，可参考 \href{https://youtu.be/-GDsMDpC3oA}{这个视频教程}。

\section{轮到你了}
若想进一步探索图~\ref{fig:SC_combined} 所示超连续谱的性质，请打开用于生成该结果的 \href{https://colab.research.google.com/drive/1HvA8F8yzEq-9fahuI4z2KhT-YhdRAXgt?usp=sharing}{notebook}，尝试下面这些实验，并解释脉冲及其频谱的演化为何会不同。在开始每个实验之前，先写下你对结果变化的预测，再与实际仿真结果对照。注意，每次实验前都应把参数重置为基线值；最简单的做法是重新点开一次链接，打开一个新的 notebook。

\begin{enumerate}

\item \textbf{没有非线性}。基线仿真中取 $\gamma>0$。把它改成 $\gamma=0$。结果是否表明非线性对脉冲时域演化有很大影响？

\item \textbf{较粗的步长分辨率}。基线仿真把 5~m 长的光纤划分为 $2^{10}=1024$ 个步长。把步数依次改为 $2^{9}$、$2^{8}$、$2^{7}$ 等，看看从什么时候开始仿真精度出现明显恶化。

\item \textbf{不同包络形状}。基线仿真使用双曲正割包络。试着把脉冲类型改成 \texttt{gauss}、\texttt{square}、\texttt{sinc}、\texttt{sqrt\_triangle} 或 \texttt{CW}。对于 \texttt{CW} 这种情况，时域输出中那种杂乱结构对应的是哪一种非线性现象？

\item \textbf{关闭自陡峭化}。基线仿真中包含自陡峭化。将其关闭，判断这一改动是否带来明显影响。

\item \textbf{取负或正的 $\alpha$}。基线仿真使用 $\alpha=0$。分别用 $\alpha=-1$~dB/m 和 $\alpha=+1$~dB/m 重跑仿真，并比较这三种情况。



\item \textbf{正的 $\betag_2$}。基线仿真使用 $\betag_2<0$。把 $\betag_2$ 的符号改成正，解释这时是哪一种非线性现象主导了脉冲演化。




\item \textbf{只保留 $\betag_2<0$}。基线仿真中 $n>2$ 时都有 $\betag_n\neq0$。把所有 $n>2$ 的 $\betag_n$ 都设为 0，重新运行，并解释为什么脉冲及其频谱的演化发生了变化。

\item \textbf{负的 $\betag_3$}。基线仿真中 $\betag_3>0$。把它改成负值。注意：为了让时间演化图显示正常，你可能还需要把时间偏移从 $-1.75$~ps 改成 $+1.75$~ps。你能解释为什么当 $\betag_3<0$ 时不会形成拉曼孤子吗？提示：用式~\ref{eq:ZDF} 计算零色散频率，并想一想当 $\betag_3$ 变号时它如何变化。可参考 \href{https://github.com/OleKrarup123/NLSE-primer/issues/2}{这个答案链接}。


\item \textbf{修改拉曼模型}。基线仿真使用式~\ref{eq:raman_basic} 来建模拉曼效应。把 \texttt{approximate\_raman\_flag} 从 \texttt{False} 改成 \texttt{True}，并用你自己的话说明：相较于完整计算式~\ref{eq:raman_basic}，使用式~\ref{eq:raman_approx_applied} 进行近似会带来什么影响。接着按 notebook 中的提示，改用式~\ref{eq:raman_new}、式~\ref{eq:Raman_exact} 或直接设 $f_R=0$。判断这些更精确的拉曼模型，是否会让结果与式~\ref{eq:raman_basic} 的预测出现实质性差别。



\end{enumerate}
